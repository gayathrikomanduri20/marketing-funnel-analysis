# Marketing Funnel Analysis Dashboard

## Objective
Analyze user journey from Leads → Customers and identify drop-off points.

## KPIs
- Total Leads: 32K
- Total Customers: 5K
- Conversion Rate: 16.4%

## Insights
- Cellular performs better than telephone.
- Conversion drops after multiple campaign attempts.
- Certain months show higher performance.

## Tools Used
- Power BI
- Python (Data Cleaning)
